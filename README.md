
Posh modules and functions
==========================

Install
-------

Add to profile:

```Powershell
    git clone https://github.com/majkinetor/posh
    cd ./posh
    "`$Env:PSModulePath += `";$pwd`"" | Out-File -Encoding ascii -Append $PROFILE
```
